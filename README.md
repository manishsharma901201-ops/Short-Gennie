# 🎬 Short-Gennie (YouTube Shorts Generator)

Auto video generator using:
- Pexels images
- Edge TTS voice
- FFmpeg video processing

## 🚀 Features
- Topic + Script input
- Auto image fetch
- Auto voice generation
- Auto video creation

## 📡 API Usage

POST /generate

### Example Request:
{
  "topic": "success mindset",
  "script": "Namaste dosto, agar aap life me successful banna chahte ho..."
}

## 🔐 Environment Variables

PEXELS_KEY = your_api_key

## 🛠 Tech Stack
- Node.js
- Express
- FFmpeg
- Edge TTS

## 📲 Output
- Final video (MP4)
- Ready for YouTube Shorts
